# liz_ty666

**Сайт Лизы** — астрология, руны, свечи, прогнозы, обучение и энергетические практики.

### О проекте

Профессиональный эзотерический сайт с акцентом на **ясность**, а не на «волшебство». 

Здесь вы можете:
- Записаться на индивидуальные разборы («Лиза, что делать?»)
- Заказать астрологические прогнозы и натальную карту
- Работать с рунами
- Заказать свечи и очищение
- Пройти обучение

**Домен:** [lizty666.ru](https://lizty666.ru)
**Telegram для записи:** [@liz_ty666](https://t.me/liz_ty666)

### Технологии

- Чистый HTML + CSS + JavaScript (Vanilla)
- GitHub Pages + кастомный домен
- Полностью статический сайт

### Структура основных страниц

- `index.html` — главная / меню услуг
- `liza-chto-delat.html` — флагманский формат «Лиза, что делать?»
- `astrology.html` — астрология и прогнозы
